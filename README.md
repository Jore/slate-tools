# @shopify/slate-tools

Tooling for Shopify themes using [slate](https://github.com/Shopify/slate).

## Installation
```bash
$ npm install @shopify/slate-tools
```

## Commands

Please refer to [slate](https://github.com/Shopify/slate) for list of commands.
